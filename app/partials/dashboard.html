Magic the Gathering API

Welcome to my Magic the Gathering API demonstration site!

The source code for this site is freely available on GitHub

The idea...

The idea for this API/site came from many different places. It started as the carrot at the end of the stick to learn developing with AngularJS (and, of course, actually playing Magic the Gathering). I wanted to see how you would go about gathering all card-information, and displaying it in an easy-to-grasp fashion.

Other than the actual importing, Laravel was the obvious choice of backend. It's simple routing and wonderful implementation of MVC, made it a joy to work with.

What started as just an API/data-access-point quickly grew into a web-app that would lend a helping hand to MtG-players (currently, mostly those that play MtG Planechaser-style).

Technology...

The frontend that you see here is built with AngularJS, and styled with Bootstrap.

The backend/API layer that you can't see, but is serving you all this magnificent information is built with Laravel.

How it works...

Once in a while, a script will run. That script will download the newest .json-file from MTGJSON, and update the database with the newest information about Magic the Gathering.

Pictures of the cards are fetched from MTGImages.
