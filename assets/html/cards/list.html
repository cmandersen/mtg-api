How to use:

Filter cards shown with the filter-dropdowns, or/and search field. Scrolling to the bottom of the list will reveal a "Show more" button that will load the next set of cards from the database.

Clicking a card will show the full card image in a pop-over. To exit the pop-over, press escape.
